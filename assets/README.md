# Assets

This directory contains project assets such as images, banners, and other media files.

## Social Banner

Place your social banner image here as `social-banner.png` (or `.jpg`, `.svg`, etc.).

Recommended dimensions:
- Width: 1280px
- Height: 640px (2:1 aspect ratio)
- Format: PNG or JPG

The banner will be displayed at the top of the README.md file.
